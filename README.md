# Github API

Dependencies

- React
- Webpack
- Babel
- Redux
- Material-UI
- styled components

## How to run?

1. Run script.
```
$ npm run start:dev
```
2. Look at in the browser.

## Build production

```
$ npm run build
```

## Run your own tunnel

1. Run npm script.
```
$ npm run start:tunnel
```
2. Open in browser and Share link in social media. :satisfied:
